## 1. 브랜드 기초

### 브랜드 에센스
> **"당신의 공간에 감성을 더하다"**

리빙짱이 존재하는 이유는 단 하나 — 일상 공간을 취향으로 채우는 경험을 만드는 것입니다.

### 브랜드 포지셔닝
감성 중심의 라이프스타일 리빙 브랜드로, 트렌드보다 무드를, 스펙보다 분위기를 우선합니다.

### 핵심 가치 (Brand Values)

| 가치 | 정의 |
|------|------|
| **무드 (Mood)** | 제품 하나로 공간의 감성이 달라진다 |
| **취향 (Taste)** | 정답 없는 나만의 스타일을 응원한다 |
| **일상 (Everyday)** | 특별한 날이 아닌, 매일을 위한 브랜드다 |

### 브랜드 퍼스낼리티 (Brand Personality)
- 따뜻하고 감각적인
- 친근하지만 세련된
- 과하지 않고 은은한
- 트렌디하되 유행을 타지 않는

## 2. 로고 사용 규정

### 로고 형태

```
한글형   리빙짱
영문형   LIVINGJJANG
조합형   리빙짱 / LIVINGJJANG
```

### 최소 사용 크기
- 디지털 : 영문 기준 최소 **80px** 이상
- 인쇄물 : 영문 기준 최소 **20mm** 이상

### 여백 규정 (Clear Space)
로고 주변에는 로고 높이의 **1/2 이상**의 여백을 확보합니다.  
다른 텍스트, 이미지, 그래픽 요소가 이 영역을 침범하지 않도록 합니다.

### 로고 사용 가능 배경

| 배경 | 권장 로고 컬러 |
|------|--------------|
| 화이트 / 라이트 계열 | 차콜 (#2C2C2C) |
| 다크 계열 | 오프화이트 (#FAF8F5) |
| 테라코타 / 머스타드 계열 | 화이트 (#FFFFFF) |

## 3. 컬러 시스템

### 메인 컬러 (Primary Colors)

| 컬러명 | HEX | 용도 |
|--------|-----|------|
| **오프화이트** `Off White` | `#FAF8F5` | 기본 배경, 전체 톤의 기반 |
| **차콜** `Charcoal` | `#2C2C2C` | 메인 텍스트, 로고 |
| **테라코타** `Terracotta` | `#C4715A` | 브랜드 포인트, 강조 요소 |

### 서브 컬러 (Secondary Colors)

| 컬러명 | HEX | 용도 |
|--------|-----|------|
| **머스타드** `Mustard` | `#D4A84B` | 시즌 강조, 포인트 |
| **웜그레이** `Warm Gray` | `#B8B2AA` | 보조 텍스트, 구분선 |
| **크림** `Cream` | `#EDE8DF` | 카드 배경, 섹션 구분 |

### 컬러 사용 비율
```
오프화이트    60%  —  배경, 여백
차콜          25%  —  텍스트, UI 요소
테라코타/머스타드  10%  —  포인트, CTA
웜그레이/크림   5%   —  보조 요소
```

## 4. 타이포그래피

### 국문 폰트

| 용도 | 폰트 | 굵기 |
|------|------|------|
| 헤드라인 | **Noto Serif KR** | Bold (700) |
| 서브 헤드라인 | **Noto Sans KR** | Medium (500) |
| 본문 | **Noto Sans KR** | Regular (400) |
| 캡션 / 보조 | **Noto Sans KR** | Light (300) |

### 영문 폰트

| 용도 | 폰트 | 굵기 |
|------|------|------|
| 헤드라인 | **Cormorant Garamond** | Semi Bold (600) |
| 서브 헤드라인 | **DM Sans** | Medium (500) |
| 본문 | **DM Sans** | Regular (400) |
| 로고 / 강조 | **DM Sans** | Bold (700) |

### 타이포 사이즈 스케일 (디지털 기준)

```
Display      40–48px   Cormorant Garamond Bold  — 메인 비주얼 카피
H1           32px      Noto Serif KR Bold       — 섹션 대제목
H2           24px      Noto Sans KR Medium      — 소제목
Body         16px      Noto Sans KR Regular     — 본문
Caption      13px      Noto Sans KR Light       — 캡션, 날짜, 보조 정보
```

### 행간 & 자간
- 국문 본문 행간 : **1.8**
- 영문 본문 행간 : **1.6**
- 헤드라인 자간 : **−0.02em** (약간 좁게)
- 캡션 자간 : **+0.05em** (약간 넓게)

## 5. 톤 앤 매너

### 브랜드 보이스 (Brand Voice)

리빙짱의 언어는 **친한 언니처럼 따뜻하고, 취향 있는 사람처럼 감각적**입니다.  
과도한 판매 언어 없이, 공간과 감성에 대한 이야기를 진솔하게 건넵니다.

### 커뮤니케이션 원칙

| 원칙 | 설명 |
|------|------|
| **진솔하게** | 과장 없이, 있는 그대로의 무드를 전합니다 |
| **감각적으로** | 감성적인 언어와 이미지로 분위기를 만듭니다 |
| **가볍게** | 무겁지 않게, 일상의 언어로 말합니다 |
| **존중하며** | 고객의 취향을 존중하고, 강요하지 않습니다 |

### 카피 스타일 예시

**좋은 예 ✅**
```
오늘 밤은 이 빛 하나로 충분해.
당신의 공간에 어울리는 무드를 찾아드릴게요.
작은 소품 하나가 하루를 다르게 만들어요.
```

**피해야 할 예 ❌**
```
지금 바로 구매하세요! 한정 수량!
최저가 보장! 놓치면 후회!
무조건 예뻐요 강추합니다!!
```

### 해시태그 사용 기준
- 핵심 브랜드 태그 : `#리빙짱` `#LIVINGJJANG` 항상 포함
- 감성 태그 : `#나만의공간` `#무드인테리어` `#홈데코` `#감성소품`
- 제품 태그 : 카테고리별 제품명 태그 병행 사용
- 과용 금지 : 게시물당 최대 **10개** 이내 유지

## 6. 이미지 & 비주얼 방향

### 포토그래피 원칙

**✅ 권장**
- 자연광 또는 자연광에 가까운 간접 조명 활용
- 오프화이트, 크림, 베이지 계열 배경
- 여백을 충분히 활용한 미니멀한 구도
- 실제 생활 공간처럼 느껴지는 라이프스타일 연출
- 질감이 느껴지는 소재 — 린넨, 세라믹, 우드, 유리

**❌ 지양**
- 과도한 보정 또는 채도 강조
- 어둡고 무거운 배경
- 복잡하고 산만한 배경 구성
- 제품만 단독으로 찍는 카탈로그식 사진

### 비주얼 무드 키워드
```
따뜻한  ·  고요한  ·  은은한  ·  정돈된  ·  감각적인
Warm  ·  Calm  ·  Subtle  ·  Curated  ·  Sensory
```

### 인스타그램 피드 구성 원칙
- 전체 피드의 컬러 톤을 오프화이트 & 웜톤으로 통일
- 제품 클로즈업 : 라이프스타일 연출 = **4 : 6** 비율 유지
- 텍스트 카드 게시물은 크림 또는 오프화이트 배경 사용
- 동일 컬러 계열 이미지를 연속 배치하는 것을 지양 (리듬감 유지)

## 7. 콘텐츠 가이드

### 콘텐츠 유형별 방향

| 유형 | 내용 | 비율 |
|------|------|------|
| 제품 연출 | 공간 속 제품 라이프스타일 사진 | 40% |
| 무드 콘텐츠 | 브랜드 감성을 담은 비주얼 / 카피 | 25% |
| 정보성 | 인테리어 팁, 제품 사용 가이드 | 20% |
| 고객 참여 | 리그램, UGC, 이벤트 | 15% |

### SNS 채널별 방향

**인스타그램 (주력 채널)**
- 피드 : 감성 라이프스타일 이미지 중심
- 스토리 : 일상적인 소통, 신제품 프리뷰, 투표/질문
- 릴스 : 공간 연출 과정, Before & After, 언박싱

**네이버 블로그**
- 제품 상세 리뷰, 인테리어 활용 가이드
- 검색 유입을 위한 키워드 중심 정보성 콘텐츠

**카카오 채널**
- 신제품 출시, 프로모션 공지
- 간결하고 직관적인 메시지 위주

## 8. 금지 사항

### 로고 관련

- ❌ 로고 비율 변형 (가로/세로 왜곡)
- ❌ 가이드라인 외 색상 적용
- ❌ 로고에 그림자, 외곽선, 그라디언트 효과 추가
- ❌ 배경과 대비가 낮아 가독성이 떨어지는 배치
- ❌ 다른 브랜드 로고와 나란히 동일 크기로 배치

### 컬러 관련

- ❌ 가이드라인에 없는 원색 계열 컬러 단독 사용
- ❌ 배경과 텍스트의 대비가 3:1 미만인 조합
- ❌ 차갑고 무채색 중심의 톤 적용

### 이미지 관련

- ❌ 지나치게 보정된 비현실적인 색감
- ❌ 브랜드 무드와 맞지 않는 배경 (형광색, 강한 패턴 등)
- ❌ 저해상도 이미지 사용 (디지털 최소 72dpi / 인쇄 최소 300dpi)

### 커뮤니케이션 관련

- ❌ 과도한 할인 강조, 자극적인 판매 언어
- ❌ 경쟁 브랜드 직접 언급 또는 비교
- ❌ 사회적 이슈에 대한 브랜드 공식 입장 표명 (내부 협의 없이)
- ❌ 고객 정보 또는 UGC 무단 사용
